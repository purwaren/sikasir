IDE SIKASIR
Sikasir Extends
- Module untuk membership
- Module untuk reading magnetic card / smart card
- Module untuk user interfaces
- Mobile Apps
- Module untuk export laporan ke PDF, Excel, Word, CSV dll
- Module untuk interkoneksi dengan aplikasi lain (API)
- Module untuk Online ( kan rencananya mau dibuat online, antar mesin terhubung ke server)
- Module untuk Feature Management / mengatur fungsionalitas mana yang boleh dan tidak boleh, berdasarkan versinya.

Ide : program sikasir dibuat beberapa versi, dikhususnkan untuk bentuk usahanya, misal:
- Retailer (Yang sekarang dikembangin)
- Food and Beverages, Restaurant (table service or quick service) a
